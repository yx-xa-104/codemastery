## Tạo Module riêng
```python
# utils.py
def slugify(text):
    return text.lower().replace(" ", "-")

def format_currency(amount):
    return f"{amount:,.0f} VNĐ"

# main.py
from utils import slugify, format_currency
print(slugify("Xin Chào"))  # "xin-chào"
```

## Package Structure
```
mypackage/
├── __init__.py
├── math_utils.py
├── string_utils.py
└── sub_package/
    ├── __init__.py
    └── helpers.py
```
