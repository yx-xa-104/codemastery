Nội dung bài học

Chào mừng các bạn đến với khóa học Lập trình Python cơ bản.

Trong bài học này, chúng ta sẽ tìm hiểu về lịch sử hình thành và các đặc điểm nổi bật của ngôn ngữ này. Python là một ngôn ngữ lập trình mạnh mẽ và dễ học.

1. Python là gì?

Python là một ngôn ngữ lập trình bậc cao, thông dịch, hướng đối tượng và đa năng. Được tạo ra bởi Guido van Rossum và lần đầu ra mắt vào năm 1991, Python triết lý thiết kế nhấn mạnh vào khả năng đọc code.

Lưu ý: Python là ngôn ngữ phân biệt chữ hoa chữ thường.
